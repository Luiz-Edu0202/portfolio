# Portfolio – Luiz Eduardo Brayner

Personal responsive portfolio with **bilingual support** (PT/EN), **light/dark theme**, and dynamic resume download based on the selected language. Built with pure HTML, CSS, and JavaScript.

## ✨ Features

- 🌐 Language toggle (Portuguese / English)
- 🌙 Light / Dark theme (persistent via localStorage)
- 📄 Resume download in the active language (PT or EN)
- 📱 Fully responsive layout (mobile‑first)
- 📧 Contact form integrated with [FormSubmit](https://formsubmit.co) – ready to send emails
- 💼 Highlighted projects: KOLEKTO, SUB!, PRIS.ME, Line Following Robot, Coffee Break

## 🛠️ Technologies

- HTML5
- CSS3 (custom properties, Flexbox, Grid)
- JavaScript (ES6)
- [Font Awesome](https://fontawesome.com) – icons
- [Google Fonts](https://fonts.google.com) – Inter font
